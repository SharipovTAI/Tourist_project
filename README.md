Проект: Привлечение и регулирование туристского потока на основе геополитических факторов
Описание
Веб-сайт с ИИ-помощником для предоставления информации о заповедниках России, их геополитическом значении и туристическом потенциале. Проект помогает регулировать туристические потоки на охраняемых территориях.

Основные функции
1) Интеллектуальный помощник на базе LLaMA 3.1
2) Семантический поиск по базе знаний о 15+ заповедниках
3) Анализ геополитических аспектов территорий
4) Ответы на вопросы о природных особенностях, истории и значении заповедников

Технологии
1) Backend: Django 4.x, Python 3.10
2) ИИ: LLaMA 3.1 8B, SentenceTransformer
3) Frontend: HTML5, CSS3, JavaScript
4) База данных: SQLite

Установка и запуск
1) Установите зависимости:

pip install django sentence-transformers numpy llama-cpp-python
Поместите модель LLaMA в папку:

D:/project_workshop/Llama_project/models/Meta-Llama-3.1-8B-Instruct-Q4_K_M.gguf
Поместите файл с базой знаний:

D:/project_workshop/tourist_project/homepage/zapovedniki_optimized_grouped.txt
Запустите Django:

python manage.py migrate
python manage.py runserver

База знаний
Содержит информацию по заповедникам:
Таймырский, Баргузинский, Кроноцкий, Столбы, Алтайский
Кавказский, Остров Врангеля, Приокско-Террасный
Дальневосточный морской, Саяно-Шушенский и другие

По каждому заповеднику:
Основные данные (год основания, площадь)
Природные особенности
Геополитическое значение
Туристический потенциал

Использование
Откройте сайт в браузере
Задайте вопросы о заповедниках России
Получите информацию с учетом геополитического контекста

Примеры вопросов:
"Расскажи про Таймырский заповедник"
"Какое геополитическое значение имеет Остров Врангеля?"
"Какие заповедники находятся на границе с другими странами?"

Для разработчиков
Проект использует RAG-архитектуру (Retrieval-Augmented Generation):
Семантический поиск по базе знаний
Извлечение релевантного контекста
Генерация ответов с ограничением на основе контекста

Проект предназначен для образовательных и исследовательских целей.
